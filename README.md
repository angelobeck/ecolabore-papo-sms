# ecolabore-papo-sms

This module allows Ecolabore to send SMS by Papo web service

Please visit <https://www.paposms.com/>

Este serviço está disponível para envio de SMS somente no Brasil.

Você precisa se cadastrar no site da Papo e comprar um pacote de mensagens para utilizar este serviço.

## Instalação

1. Coloque a pasta "PapoSMS" na pasta de componentes do seu ecolabore.
2. No painel administrativo vá em Ferramentas -> Componentes e habilite o PapoSMS.
3. Em seguida vá em Serviços -> Envio de mensagens por SMS -> Configurar conta padrão para envio SMS
4. Preencha o formulário com o email e a senha da sua conta Papo e salve o formulário.

Duas novas páginas estarão disponíveis: "Testar o envio de mensagens" e "Saldo", onde você poderá testar o funcionamento do serviço e verificar informações de sua conta Papo, respectivamente.

Os números de telefone precisam iniciar com "+55", porque somente são aceitos números do Brasil.
